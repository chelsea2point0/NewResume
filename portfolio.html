Invoice Manager

This website is an invoice manager site for a clothing store. Customers can enter the items they are purchasing and it will add the item to the invoice. Customers can also remove unwanted items from the invoice. If a duplicate item code is entered, the program will prompt the user to either add the current quantity to the existing quantity or cancel and return to the form. This page also adheres to unobtrusive JavaScript principles and provides a noscript static HTML page to users who do not have JavaScript enabled.

Synapse Airlines


This webpage uses Knockout JS to allow the client to create seat reservations with Synapse Airlines. Depending upon client selections, the line total adjusts and the sum of all reservations is calculated into Total Surcharge.
